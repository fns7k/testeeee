# Frontend E-commerce Platform

This is the frontend part of the E-commerce platform project. It is built using React and TypeScript, providing a user-friendly interface for customers to browse products, manage their cart, and complete the checkout process.

## Features

- **Home Page**: Displays featured products and categories.
- **Product Page**: Detailed view of individual products with reviews and ratings.
- **Checkout Page**: Allows users to enter their address and select payment options.
- **Admin Panel**: Manage products, view orders, and upload CSV files for bulk product updates.
- **Advanced Search**: Includes an autocomplete search bar and product suggestions.
- **Review System**: Customers can leave reviews with star ratings and upload photos.

## Getting Started

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the frontend directory**:
   ```
   cd ecommerce-platform/frontend
   ```

3. **Install dependencies**:
   ```
   npm install
   ```

4. **Run the application**:
   ```
   npm start
   ```

## Folder Structure

- `public/`: Contains the main HTML file.
- `src/`: Contains all the source code for the application.
  - `components/`: Reusable components like Header, Footer, ProductCard, etc.
  - `pages/`: Different pages of the application.
  - `hooks/`: Custom hooks for managing state and side effects.
  - `services/`: API service functions for backend communication.
  - `styles/`: Global styles for the application.
  - `types/`: TypeScript types used throughout the application.
